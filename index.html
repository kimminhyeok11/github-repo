GitHub Pages Deployer

정식 GitHub Pages 호스팅 자동화 도구

토큰은 GitHub에 저장되지 않으며, 배포 시에만 사용됩니다.

이 도구는 어떻게 작동하나요?

이 도구는 GitHub의 공식 API를 사용하여, 가장 안정적인 방식으로 아래 4단계 배포 과정을 자동으로 처리합니다.

  • [개선] 저장소 생성 및 초기화: 텅 빈 저장소가 아닌, `README.md` 파일이 포함된 '준비된' 저장소를 생성하여 타이밍 오류를 원천 차단합니다.
  • [개선] 파일 직접 업로드: 복잡한 절차 대신, 가장 간단하고 확실한 `PUT /contents` API를 사용하여 `index.html` 파일을 저장소에 직접 생성합니다.
  • Pages 기능 활성화: 업로드된 저장소의 GitHub Pages 호스팅 기능을 켭니다.
  • URL 확인: GitHub 서버의 빌드가 완료되고, 실제 웹사이트 주소가 활성화될 때까지 기다린 후 최종 결과를 보여줍니다.
